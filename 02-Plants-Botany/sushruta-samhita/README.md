# Sushruta Samhita

**Author:** various
**Era / Date:** c. 6th c. BCE–4th c. CE
**Origin / Tradition:** India (Sushruta)
**Original language:** Sanskrit (original)
**Modern discipline(s):** Ayurveda, surgery, botany

## Summary

The Sushruta Samhita — the foundational Ayurvedic surgery text, with extensive plant-based pharmacopoeia.

## Why it matters

The other root text of Ayurveda; the primary source for classical Indian surgical and herbal knowledge.
