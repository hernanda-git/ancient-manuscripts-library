# Ajaib Literature

**Author:** various
**Era / Date:** 8th–16th c. CE
**Origin / Tradition:** Persia/Islamicate
**Original language:** Persian/Arabic (original)
**Modern discipline(s):** Cosmographical wonders, the unseen

## Summary

The 'aja'ib' (wonders) genre — Persian cosmographies (e.g. 'Aja'ib al-Dunya, 'Aja'ib al-Buldan) describing marvels, lands, and beings of the unseen.

## Why it matters

The literary framework through which the Islamic world mapped the marvellous and the jinn-inhabited world.
