# The Holy Qur'an (القرآن)

**Author / origin:** Islamic primary scripture — revealed 610–632 CE, codified 7th c.
**Original language:** Arabic (Uthmani text)
**Translation included:** Pickthall (Marmaduke Pickthall, 1930 — public domain)

## Summary
The central religious text of Islam. It repeatedly affirms the existence of the **jinn** (Sura al-Jinn, 72, is named for them), angels, and the unseen (*al-ghayb*), making it the primary source for the entire 05-Jinn-Unseen tradition.
